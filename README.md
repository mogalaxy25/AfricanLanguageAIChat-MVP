How to Run Locally
====================================================================

# 1. Clone the repo
$ git clone git@github.com:mogalaxy25/AfricanLanguageAIChat-MVP.git
$ cd AfricanLanguageAIChat-MVP

# 2. Get Flutter dependencies
$ flutter pub get

# 3. Run on device or emulator
$ flutter run
